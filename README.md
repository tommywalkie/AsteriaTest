# AsteriaTest

Technical test intended to showcase projects, their technical challenges and their biological models

## Requirements

- Node.js 18+
- pnpm

## Install

```bash
nvm use # If you have nvm installed, set the right Node.js version.
pnpm install
```

## Run

```bash
pnpm dev
```

## Test

```bash
pnpm test
```

## Typecheck

```bash
pnpm typecheck
```

## Build

```bash
pnpm build
```
